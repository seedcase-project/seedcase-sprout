Machine-Readable Name
The name that you would like the column/variable to have going forward. This name can't include spaces or special characters.
Display Name
The name that can be used when you want to show the data in a more 'human readable' way. An example could be the variable b60, which you could give the more descriptive display name Blood taken after 60 minutes.

A short description of the column/variable, including things like when a sample is taken, the unit, etc.
